🩺 Patient Monitor System – Project Description

This project is a simple and smart web-based healthcare assistant built using pure HTML, CSS, and JavaScript . The system allows users to upload an image of a hand injury, and based on the image (filename logic), it identifies the type of injury and provides useful medical guidance.

📸 Once an image is uploaded:

The system previews the image
It analyzes the file name to detect injury type (like burn, cut, bruise, rash, infection)
Displays detailed results including:
🧠 Injury name
📖 Description
💊 Treatment steps
🧴 Recommended products
⚠️ When to consult a doctor

💡 The system is designed to be lightweight, clean, and privacy-friendly — all processing happens in the browser, with no server upload.

📡 Additionally, it supports integration with an ESP32 camera module, allowing real-time image capture and analysis from hardware devices.
