    ESP32 Camera Integration

    This interface can receive images directly from an ESP32 camera module. The ESP32 can capture patient images and send them to this page for analysis.

    How it works:

    1. ESP32 captures image via camera module

    2. Image is sent to this web application

    3. System analyzes and displays injury information

    4. Care suggestions are provided
